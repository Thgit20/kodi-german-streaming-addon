# Installation des German Streaming Addons über Repository

## Methode 1: Direkt über Repository-URL (Empfohlen)

### Schritt 1: Repository hinzufügen
1. Öffne Kodi
2. Gehe zu **Einstellungen → Addon Manager → Addon-Repository → ... (Benutzerdefiniert installieren)**
3. Gib folgende URL ein:
   ```
   https://github.com/Thgit20/kodi-german-streaming-addon/raw/main/
   ```
4. Klicke OK

### Schritt 2: Addon installieren
1. Gehe zu **Addon Manager → Video-Addons**
2. Wähle **German Streaming Repository**
3. Klicke **Installieren**

### Schritt 3: German Streaming Addon installieren
1. Gehe zu **Add-ons → Video-Addons**
2. Suche nach **German Streaming**
3. Klicke **Installieren**

## Methode 2: Manuell über Zip-Datei

1. Lade die neueste Version herunter:
   - Gehe zu: https://github.com/Thgit20/kodi-german-streaming-addon/releases
   - Lade die `.zip` Datei herunter

2. In Kodi:
   - Gehe zu **Add-ons → Addon-Manager → Aus ZIP-Datei installieren**
   - Navigiere zur heruntergeladenen Datei
   - Klicke OK

## Methode 3: Aus Verzeichnis

1. Klone das Repository:
   ```bash
   git clone https://github.com/Thgit20/kodi-german-streaming-addon.git
   ```

2. Kopiere auf Android:
   ```
   /storage/emulated/0/Android/data/org.xbmc.kodi/files/.kodi/addons/
   ```

3. Starte Kodi neu

## Repository-URL zum Manuellen Hinzufügen

Falls die automatische Methode nicht funktioniert, nutze diese URL:
```
https://github.com/Thgit20/kodi-german-streaming-addon/raw/main/
```

## Updates

Sobald das Repository hinzugefügt ist, prüft Kodi automatisch auf Updates. Du kannst auch manuell prüfen:
1. Gehe zu **Add-ons → Addon-Manager**
2. Wähle **German Streaming**
3. Klicke **Update** (falls verfügbar)

## Troubleshooting

### Repository wird nicht gefunden
- Überprüfe deine Internetverbindung
- Stelle sicher, dass die URL korrekt ist
- Versuche später erneut

### Addon-Installation schlägt fehl
- Stelle sicher, dass genug Speicher vorhanden ist
- Kodi neu starten
- Versuche die manuelle ZIP-Installation

### Updates funktionieren nicht
- Gehe zu **Einstellungen → System → Add-ons**
- Aktiviere **Automatische Update-Prüfung**
- Starte Kodi neu
